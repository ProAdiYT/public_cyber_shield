# Security Policy

## Supported Versions

We currently support and maintain security updates for the following versions:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0.0 | No        |

## Reporting a Vulnerability

We take the security of Citizen Cyber Shield seriously. If you find a security vulnerability, please do not report it publicly (via GitHub issues or PR comments). Instead, follow these steps:

1. **Email us**: Send a detailed report to security@citizencybershield.com.
2. **Provide Details**: Include a description of the vulnerability, a proof of concept (PoC), and instructions on how to reproduce it.
3. **Wait for response**: We will acknowledge your email within 48 hours and work on a fix.
4. **Coordinated Disclosure**: We ask you to give us up to 30 days to release a security patch before disclosing the vulnerability publicly.
